# Tiny World

Tiny World is a lightweight system-tray AI assistant that reads text from your clipboard and provides instant answers or code. Trigger requests with hotkeys, get results copied back to the clipboard, and receive desktop notifications—all while running quietly in the background.

## Features

- **Clipboard AI Assistant**
  - Reads text from clipboard automatically.
  - Supports code generation (`s+c`) and general questions (`s+q`).
  - Results are copied back to clipboard instantly.

- **Hotkey Support**
  - `s+c` → Generate code.
  - `s+q` → Answer questions.
  - Runs in background threads for responsiveness.

- **System Tray Integration**
  - Runs quietly in the system tray.
  - Custom blue-and-white icon.
  - Exit option in tray menu.

- **Desktop Notifications**
  - Shows success/failure notifications after processing requests.

- **API Integration**
  - Connects to OpenRouter AI (`google/gemini-2.0-flash-exp:free`) for responses.
  - Uses system prompts for smarter outputs.

- **Robust Logging**
  - Logs events and errors to `tiny_world_debug.txt`.
  - Handles errors in imports, API calls, clipboard access, and hotkeys.

- **Lightweight and Efficient**
  - Minimal dependencies.
  - Low CPU usage; runs in the background.
